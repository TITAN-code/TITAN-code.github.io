Welcome to the TITAN webpage!

TITAN (elecTric fIeld generaTion And maNipulation) is being developed by:

Thijs Stuyver, Jing Huang, Dibyendu Mallick, Sason Shaik

The current version is TITAN - v2.0

TITAN - v2.0 can be downloaded by clicking HERE (.tar.gz) or HERE (.zip).

A manual, input templates and examples are included in the main folder.

In the latest version, the code has been redesigned from scratch according to

object-oriented principles and has been made compatible with PYTHON3.

Please refer to TITAN as follows:

Thijs Stuyver, Jing Huang, Dibyendu Mallick, Sason Shaik, J. Comput. Chem. 2020, 41, 74-81
